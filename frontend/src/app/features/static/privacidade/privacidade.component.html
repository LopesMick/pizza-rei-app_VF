Política de Privacidade

Esta aplicação utiliza armazenamento local do navegador (localStorage) para simular funcionalidades como cadastro de usuário, carrinho e histórico de pedidos.

O que é armazenado (simulação)

  • Dados do cadastro preenchidos no formulário
  • Itens adicionados ao carrinho
  • Pedidos finalizados para histórico local

Importante

  • Nenhuma informação é enviada para servidores externos neste estágio do projeto.
  • Os dados podem ser apagados a qualquer momento limpando o cache do navegador.

Em uma versão integrada ao backend, esta política deve ser revisada para contemplar regras reais de tratamento e segurança de dados.
